Comandos GIT:

ls (lista o que tem no local/pasta)
cd (muda o diretório/pasta)
pwd (mostra o caminho absoluto de onde estou)
ls -a (arquivos que começam com ponto; ./ e ../)
mkdir (cria uma nova pasta)
cat comandos (mostra o que tem dentro do arquivo)
git init . (cria um repositório git vazio)
rm -rf .git/ (apaga a pasta git sem pedir autorização)
