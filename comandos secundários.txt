Esse arquivo contém comandos secundários de git:

git checkout -b <nome da branch> (cria uma nova branch; como se tivesse copiado o nosso projeto para uma nova pasta e agora está trabalhando em outra pasta)
git checkout <nome da branch> (quando já existe a branch)
git branch (mostra em qual branch está)
git merge <nome da branch> (traz as informações da branch secundária para a main)
git branch -D <nome da branch> (deleta a branch)
git remote add origin
git push origin main
touch README.md
git pull origin main
